# Phone Book#
## A simple Phone Book app for storing your contacts ##
- - - -

### Using ###
* React
* Redux
* NodeJS
* MongoDB
* Mongoose

### Included ###
* webpack
* jQuery
* Foundation framework
* SCSS
* source maps
* browser history


### Global dependencies ###
* webpack 

### Requirements ###
* Node.js
* GitBash
* Mongo

### Starting the app in local environment ###
1. run npm install
2. run mongod from terminal
3. run webpack (this will generate bundle file)
4. run node server.js

### Linter ###
* ESLint

### API endpoint ###
* **save data** 
* "/contacts"
* **retrieve data**
* "/contacts"
* **delete data**
* "/contacts/id"
